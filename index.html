Portfolio

List of completed game projects that I worked on

Tanki

Tanki is a 2D Tank shooter game made in Unity with 20 levels and a hand-drawn like artstyle. It was a group project where I was the main programmer. I was responsible for game mechanics, UI and level design. Repositorium is on: Github. Downloadable from itch.io.

Nuke-o-cat

Nuke-o-cat is a 2-player 3D arcade game with 2 differnet stages made in Unity on 44h RPG Game Jam. I was the main programmer in the team. I was responsible for game mechanics, main menu and SFXs. Repositorium is on: Github. Playable from itch.io.

Pre2hou

Pre2hou is a Touhou themed 2D bullet-hell shooter made in C++ with SFML library. It's a solo project with 2 levels with bosses, where I programmed all game mechanics and designed levels. Repositorium is on: Github. Downloadable from Github.

Plaint

Plaint is a 2D puzzle game made in Godot on 48h Mini Game Jame. It was a group project where I was responsible for movement mechanics and level design. Repositorium is on: Github. Playable from itch.io.
